# VIO Documentation

Welcome to the VIO project documentation.

## Quick Start

This documentation site is built with MkDocs and supports Mermaid diagrams.

## Example Mermaid Diagram

```mermaid
graph TD
    A[Start] --> B{Is it working?}
    B -->|Yes| C[Great!]
    B -->|No| D[Debug]
    D --> A
```
